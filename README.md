# COPD Detection using Machine Learning

This project predicts Chronic Obstructive Pulmonary Disease (COPD)
using machine learning models trained on medical data.

Algorithms Used
- Logistic Regression
- K-Nearest Neighbors
- Decision Tree
- Random Forest
- Support Vector Machine

Workflow
1. Data preprocessing
2. Exploratory Data Analysis
3. Model training
4. Model evaluation

Result
Random Forest achieved the best accuracy among all models.
